Nomenclature

Le system de nomenclature de class CSS est un dérivé du système BEM (Block--Element--modifier)
Il utilise l'Element en suffixe qui est inspiré de la sémantique des balise html.
Le suffixe est séparer par 2 tirets du nom du bloc --.
Le nom du block peut être composé avec des underscore.

Liste des suffixes element

  • h : pour le titre principale
  • l : pour une liste de block
  • i : pour l'item d'une liste de block
  • a : pour le lien principal du block
  • img : image du block
  • metas : pour le wrapper listant les metas de l'item (date, auteur, catégories, …)

D'autres suffixes pourront être créés pour les besoins du projet

Exemples

Block de menu principal

		
<nav class="main_nav">
	<ul class="l--main_nav">
		<li class="i--main_nav">
			<a href="#" class="a--main_nav">Link</a>
		</li>
	</ul>
</nav>

Liste d'articles

Les prefixes peuvent être cumulés, essentiellement dans listes de résultats

		
<sections class="articles">
	<ul class="l--articles">
		<li class="i--articles">
			<h2 class="h--i--articles">Titre de l'article</h2>
			<div class="img--i--articles">
				<img src="#" alt="#" />
			</div>
			<div class="metas--i--articles">
				Auteur + Date
			</div>
		</li>
		<li>…
	</ul>
</sections>
